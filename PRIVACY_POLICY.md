# Privacy Policy

## VoiceUp

This Privacy Policy explains how the VoiceUp extension collects, uses, and protects user data.

### Data Collected
- Audio recordings from mission tasks in Google Classroom
- Page content from Google Classroom necessary to identify mission activity and progress
- Local settings and session state required for extension functionality

### Purpose of Data Collection
- To generate progress reports for language practice missions
- To associate audio recordings with mission activity and student progress
- To support extension functionality and improve user experience

### Data Usage
- Audio data is used only to generate and display mission progress reports
- Data is not sold or shared with third parties outside of the approved use of the extension
- Data is not used for credit scoring, lending, or unrelated purposes

### Data Sharing
- VoiceUp does not sell user data
- VoiceUp does not transfer user data to unauthorized third parties
- Data is used only for the stated purpose of the extension

### Development and Testing
- Local development URLs (`http://localhost:5174`, `http://localhost:8080`) are used only for testing
- No remote code is loaded beyond what is included in the extension package

### Contact
If you have questions about this policy, please contact the project maintainer through the repository.
